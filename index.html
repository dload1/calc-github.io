Features

  1. This calculator functions in between a traditional calculator and a computer. In the upper display you may enter the calculation in equation mode like
    52 - 89² + √67 + sin(89) - ln(87).
    Press the = key to get the result in the lower display field.
  2. Select either the simple or scientific option and the corresponding calculator will be displayed
  3. For trignometric functions (sin, cos, tan) the default angle is in radians. When degree is selected, the calculator will automatically insert the multiplication factor for conversion into radians. You may enter the degrees angle afterwards.
  4. Similarly, for inverse trignometric functions the multiplication factor for conversion of angle into degrees will be inserted automatically.
  5. Note that when any function key is pressed the corresponding Javascript function is displayed in the upper equation display. Don't be dismayed by that. e.g. sin(56) will be displayed as Math.sin(56). Don't forget to close the bracket otherwise calculation will not be done.
  6. Use the 2nd/1st function key to toggle between functions

Limitation

  • Javascript math is not accurate for floating point numbers. So sin(π) may yield an infinitesimally small (O(10e-16)) but non-zero number occassionally.
  • Ensure that the equation entered is syntactically correct. All the opened brackets must be closed.
